---
name: weekly-report
description: Generate weekly e-commerce performance report
---

Using the `dtc_dashboard` tool for the last 7 days, generate a comprehensive weekly performance report. Include:

1. **Revenue Overview**: Total revenue, order count, AOV with week-over-week comparison
2. **Email Performance**: Top 5 campaigns by revenue, open rates, click rates
3. **Flow Performance**: Top 5 flows by revenue, conversion rates
4. **Subscriber Health**: List growth, engagement tiers
5. **Email Attribution**: Email revenue as % of total, flow vs campaign split
6. **Actionable Recommendations**: 3 specific, data-backed suggestions for improvement

Flag any metrics outside healthy ranges using the DTC Metrics Reference benchmarks.
